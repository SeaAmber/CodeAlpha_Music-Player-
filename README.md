JavaScript Music Player

Tech Stack: HTML, CSS, JavaScript

I added media queries for both mobile and tablet. It is responsive on desktop,tablet, and mobile.

This music player plays Christian worship music.

You can click the previous, play, and next button to go back to the previous song, go to the next song, or pause a song.

When you click previous, play, or next button, you will see it focused, unless you click somewhere else then the focus will stay on these buttons. If you click outside of these buttons, it still will  play the music, unless you paused the song.


There is a progress bar, which is the first bar, where you can see the slider moving as the song is playing. You can move the first slider(which is the progress bar.) if you want to go farther in the song.


You can move the volume slider to make the volume go up or down. 
If you don't want any sound, you can turn the slider all the way down.


This music player has autoplay where it automatically goes to the next song after each song is done. It goes back to first song and replay the first song again after playing the last song.


The features: Play, Pause, Next, auto play, progress bar, volume, playlist where it shows all the songs on the player.
